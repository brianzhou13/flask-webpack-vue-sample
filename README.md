# flask-webpack-vue-sample
a example of using flask-webpack with vuejs and hot module replacement.

### Usage

for frontend (vuejs)

``` bash
$ yarn
$ yarn run start
```
or ..
``` bash
$ npm install
$ npm run start
```

for backend (flask app)


``` bash
$ cd flask
$ virtualenv venv
$ source ./venv/bin/activate
$ pip install flask flask-webpack
$ python app.py
```
